# Opportunity Desk – Frontend Mock

This repository contains a Vite + React (TypeScript) frontend prototype for an internal "Opportunity Desk" workspace that helps a hiring team collaborate on a lead product designer search. The UI is static and does not integrate with any backend services.

## Getting started

```bash
cd frontend
npm install
npm run dev
```

The development server runs on [http://localhost:5173](http://localhost:5173). The UI is responsive down to tablet viewports and showcases:

- Summary metrics for the open role.
- Filter chips to illustrate configurable candidate filters.
- A recommended candidate grid with skills, availability, and match score.
- A pipeline board showing the status of each hiring stage.
- A live activity feed and quick action shortcuts for recruiters.
